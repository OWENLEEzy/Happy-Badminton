## Data Methodology

- **Base data scraped from:** BadmintonRanks.com (as of 2026-03-07)
- **AE prelim wins:** Each SF player won 3 matches (R1, R2, QF) to reach the SF.
  Since these were not yet in the BadmintonRanks database on SF day, 3 W results
  were prepended to each player's recent match sequence, and the oldest 3 matches
  were dropped from the 20-match window.
- **Streak sign convention:** + = winning streak, - = losing streak
